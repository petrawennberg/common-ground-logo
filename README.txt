COMMON GROUND — LOGO KIT
========================

WHAT'S INSIDE
  svg/   Vector, infinitely scalable. Wordmark is outlined (no font needed).
         Best for designers, web, large print, signage.
  png/   Transparent background, high resolution. Drop into docs, slides,
         Canva, social. Use these if you're not sure what to use.
  pdf/   Vector, print-ready. Hand to a printer for flyers, banners, swag.

LOCKUPS (each comes in color / black / white)
  mark*                         The circle symbol alone — avatars, favicons, stamps.
  lockup-horizontal*            Symbol + "Common Ground". The default logo.
  lockup-stacked*               Symbol above "Common Ground". Square-ish spaces.
  lockup-horizontal-altadena*   "Common Ground · Altadena" — when location helps.
  lockup-stacked-altadena-left* Mark above "Common Ground" / "Altadena", left-aligned.
  lockup-stacked-altadena*      Same, centered.

AVATARS (square, background baked in — for profile pictures)
  avatar-forest   White mark on forest green. The primary avatar.
  avatar-cream    Color mark on cream.
  avatar-sage     White mark on sage green.
  avatar-white    Color mark on white.
  Sized so the mark stays inside the circular crop social platforms apply.

COLOR VERSIONS
  *-color   Full brand color. Use on white or light (cream) backgrounds.
  *-black   Single-color black. Faxes, stamps, engraving, one-color print.
  *-white   Reverse / knockout. Use ONLY on dark backgrounds (e.g. forest green).
            (The white art is invisible on white — that's expected.)

BRAND COLORS
  Forest   #2E3D26      Sage      #7A8A6E
  Cream    #FAF6F0      Sage dk   #5A6953
  Coral    #E5905A

TYPEFACES
  Wordmark : DM Serif Display
  Text/UI  : DM Sans
  (Both free from Google Fonts. The logo files don't require them.)

USING IT WELL
  • Keep clear space around the logo — at least the height of the circle mark.
  • Don't recolor, stretch, rotate, or add effects.
  • Minimum size: keep the wordmark legible (~24px tall on screen).
  • On photos or busy backgrounds, use the white version over a dark area.
